provenance — they reference crate layouts, feature flags, and
disposition tables that **predate** the current architecture and are
**not** authoritative for what the code does today.

For user-facing design, read:

- [`../PHILOSOPHY.md`](../PHILOSOPHY.md) — design principles
- [`../design-v0.1.0.md`](../design-v0.1.0.md) — type-level spec (current)
